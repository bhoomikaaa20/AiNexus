What is Supervised Learning?

Supervised learning is a machine learning approach that's defined by its use of labeled datasets to train algorithms that to classify data or predict outcomes accurately. As input data is fed into the model, it adjusts its weights until the model has been fitted appropriately.

Key Concepts

  • Classification: Uses an algorithm to accurately assign test data into specific categories (e.g., separating apples from oranges).
  • Regression: Used to understand the relationship between dependent and independent variables to forecast numerical values (e.g., predicting house prices).

A classic example of supervised learning is training an algorithm to recognize pictures of cats. You feed the algorithm thousands of pictures labeled "cat" and thousands of pictures labeled "not a cat". Over time, it learns the features of a cat and can identify one in a new, unlabeled picture.
