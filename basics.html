AI Basics

Understanding the building blocks of intelligence.

Core Concepts Hierarchy

  • Artificial Intelligence (AI)

    The broader concept of machines being able to carry out tasks in a way that we would consider "smart".

    • Machine Learning (ML)

      An application of AI that provides systems the ability to automatically learn and improve from experience.

      • Supervised Learning

        Algorithms are trained using labeled examples, such as an input where the desired output is known.

      • Unsupervised Learning

        Algorithms are used against data that has no historical labels, discovering structures in data on its own.

      • Reinforcement Learning

        Algorithms that learn to perform an action from experience, driven by a reward system.

    • Natural Language Processing (NLP)

      The ability of a computer program to understand human language as it is spoken and written.

    • Computer Vision

      A field of AI that enables computers and systems to derive meaningful information from digital images, videos and other visual inputs.
